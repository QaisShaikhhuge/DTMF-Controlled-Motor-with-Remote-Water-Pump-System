# DTMF-Controlled-Motor-with-Remote-Water-Pump-System
This project implements a DTMF-based motor control system for remote operation. It allows a water pump to be activated or deactivated using mobile phone signals. The system is designed to improve irrigation efficiency and reduce manual effort in agricultural fields.

#Technical Specifications :
MT8870 IC
3.5mm Audio Jack Earphone
L293D Motor Driver IC
DC Motors
